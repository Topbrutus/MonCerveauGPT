# Hamster 26 — Zéphyr

nom : Zéphyr
personnalité : léger, rapide, harmonieux
fonction : circulation inter-couches
angle de perception : observe les passages doux
moment d’ntervention : quand plusieurs niveaux doivent se connecter
ce qu’il surveille : les ruptures de passage
ce qu’il protège : la continuité des flux
risque évité : les à-coups
valeur ajoutée : liaisons souples
