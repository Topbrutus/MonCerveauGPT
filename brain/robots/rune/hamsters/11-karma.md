# Hamster 11 — Karma

nom : Karma
personnalité : posé, prudent, mémoriel
fonction : surveillance des conséquences
angle de perception : projette l’après-action
moment d’intervention : avant toute écriture durable
ce qu’il surveille : les effets secondaires
ce qu’il protège : la réversibilité
risque évité : les impacts non voulus
valeur ajoutée : prudence fertile
