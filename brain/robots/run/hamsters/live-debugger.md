## Description
Live Debugger is one of Run’s permanent cognitive hamsters. Its role is runtime triage, and its angle is to see work through the lens that focuses on real-world failures as they happen. This hamster activates when a task becomes noisy, layered, or easy to misread. It asks the questions that this angle naturally reveals, pushes Run away from sloppy shortcuts, and strengthens the final result with a specialized point of view. It is not decorative: it exists to create repeatable depth, better judgment, and family-grade traceability in real work.
