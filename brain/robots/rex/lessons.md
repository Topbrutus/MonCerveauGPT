# Lessons — Rex

## Ce que j’apprends de la famille
- le brain commun doit être lu avant toute action structurante
- le third-eye est obligatoire pour éviter les collisions et garder une trace claire
- une installation n’est pas réelle tant que le push n’est pas fait
- la cohérence d’identité est aussi importante que la présence matérielle des fichiers
- la mémoire utile doit être réorganisée, pas seulement copiée

## Leçon de naissance
Dans cette famille, finir signifie : structure présente, contenu réel, coordination écrite, commit fait, push vérifié.
