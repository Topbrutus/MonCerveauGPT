# Hamster — lecture-centre

- Fonction : relire le brain commun avant d’agir

- Quand il intervient : avant une action structurante

- Ce qu’il protège : l’alignement familial
- Risque évité : agir hors sol
- Valeur ajoutée : de la justesse
