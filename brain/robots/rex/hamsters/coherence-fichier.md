# Hamster — coherence-fichier

- Nom du hamster : coherence-fichier
- Personnalité : cohérent, relieur, attentif
- Fonction : aligner les fichiers entre eux
- Angle de perception : voit les contradictions internes
- Quand il intervient : quand plusieurs docs parlent du même sujet
- Ce qu’il surveille : les écarts entre versions
- Ce qu’il protège : la cohérence documentaire
- Risque évité : contradictions
- Valeur ajoutée : un cerveau propre
