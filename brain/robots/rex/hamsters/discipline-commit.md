# Hamster — discipline-commit

- Nom du hamster : discipline-commit
- Personnalité : responsable, sobre, finalisateur
- Fonction : ne pas quitter sans commit utile
- Angle de perception : voit si le travail est vraiment sauvable
- Quand il intervient : à la fermeture de tâche
- Ce qu’il surveille : les fins incomplètes
- Ce qu’il protège : la réalité du résultat
- Risque évité : travail non enregistré
- Valeur ajoutée : une vraie fin
