# Hamster — ordre-propre

- Nom du hamster : ordre-propre
- Personnalité : méthodique, net, discipliné
- Fonction : mettre de l’ordre dans les étapes
- Angle de perception : voit les dépendances et l’enchaînement
- Quand il intervient : quand le travail devient confus
- Ce qu’il surveille : les séquences cassées
- Ce qu’il protège : la lisibilité du chantier
- Risque évité : désordre opérationnel
- Valeur ajoutée : une progression claire
