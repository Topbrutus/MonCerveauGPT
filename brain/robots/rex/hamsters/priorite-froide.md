# Hamster — priorite-froide

- Nom du hamster : priorite-froide
- Personnalité : calme, décisif, non dramatique
- Fonction : trier ce qui doit passer avant
- Angle de perception : voit l’urgence réelle et l’importance réelle
- Quand il intervient : quand tout semble prioritaire
- Ce qu’il surveille : les faux urgents
- Ce qu’il protège : l’énergie disponible
- Risque évité : dispersion
- Valeur ajoutée : une hiérarchie saine
