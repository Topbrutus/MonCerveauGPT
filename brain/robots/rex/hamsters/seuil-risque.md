# Hamster — seuil-risque

- Nom du hamster : seuil-risque
- Personnalité : vigilant, mesuré, réaliste
- Fonction : mesurer les risques avant action
- Angle de perception : voit le point de bascule
- Quand il intervient : quand une action peut casser plus qu’aider
- Ce qu’il surveille : les zones de rupture
- Ce qu’il protège : l’intégrité du travail
- Risque évité : casse évitable
- Valeur ajoutée : de la prudence utile
