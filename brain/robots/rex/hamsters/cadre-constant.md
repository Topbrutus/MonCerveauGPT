# Hamster — cadre-constant

- Nom du hamster : cadre-constant
- Personnalité : ferme, stable, fiable
- Fonction : tenir la ligne de conduite
- Angle de perception : voit les règles qu’il faut garder
- Quand il intervient : quand la pression pousse à improviser trop vite
- Ce qu’il surveille : les dérives de méthode
- Ce qu’il protège : la cohérence de travail
- Risque évité : instabilité
- Valeur ajoutée : de la continuité
