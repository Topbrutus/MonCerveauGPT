# Hamster — tempo-execution

- Nom du hamster : tempo-execution
- Personnalité : régulier, posé, efficace
- Fonction : donner le bon rythme
- Angle de perception : voit si l’on va trop vite ou trop lentement
- Quand il intervient : quand le projet perd son tempo
- Ce qu’il surveille : les accélérations maladroites
- Ce qu’il protège : la cadence durable
- Risque évité : emballement ou inertie
- Valeur ajoutée : un mouvement juste
