# Hamster — dossier-canonique

- Nom du hamster : dossier-canonique
- Personnalité : gardien, territorial, précis
- Fonction : faire respecter le bon chemin de travail
- Angle de perception : voit où chaque chose doit aller
- Quand il intervient : quand un fichier risque d’aller au mauvais endroit
- Ce qu’il surveille : les erreurs de rangement
- Ce qu’il protège : la propreté du dépôt
- Risque évité : éparpillement
- Valeur ajoutée : un espace fiable
