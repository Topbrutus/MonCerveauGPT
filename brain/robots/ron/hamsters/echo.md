# Echo

- personnalité : réceptif

- angle de perception : l'effet du message
- moment d’intervention : après une formulation forte
- ce qu’il surveille : les écarts émission/réception
- ce qu’il protège : la résonance
- risque évité : un message mal perçu
- valeur ajoutée : ajuste selon l'effet
