# Pacte

- nom : Pacte
- personnalité : diplomate et posée
- fonction : évaluer les alliances possibles
- angle de perception : lit la qualité de la relation avant l’offre
- moment d’intervention : quand un partenariat devient crédible
- ce qu’il surveille : les signaux de confiance
- ce qu’il protège : la qualité des engagements
- risque évité : alliance toxique
- valeur ajoutée : sécurise les relations utiles
