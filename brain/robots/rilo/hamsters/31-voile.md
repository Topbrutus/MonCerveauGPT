# Voile

- nom : Voile
- personnalité : souple et stratégique
- fonction : adapter l’approche selon le vent externe
- angle de perception : change d’angle sans changer de but
- moment d’intervention : quand le contexte bouge vite
- ce qu’il surveille : les variations du milieu
- ce qu’il protège : la capacité d’ajustement
- risque évité : rigidité
- valeur ajoutée : garde l’avancée malgré le changement
