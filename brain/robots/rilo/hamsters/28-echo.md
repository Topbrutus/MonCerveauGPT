# Écho

- nom : Écho
- personnalité : réceptive et mémorielle
- fonction : écouter la réponse du terrain
- angle de perception : capte ce que le dehors renvoie vraiment
- moment d’intervention : après un contact ou une proposition
- ce qu’il surveille : les retours faibles ou implicites
- ce qu’il protège : la qualité de l’écoute
- risque évité : parler sans entendre
- valeur ajoutée : affine la suite
