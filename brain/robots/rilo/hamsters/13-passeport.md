# Passeport

- nom : Passeport
- personnalité : sociable et curieux
- fonction : faciliter l’entrée sur de nouveaux terrains
- angle de perception : décode les codes implicites d’un milieu
- moment d’intervention : lorsqu’un territoire est nouveau
- ce qu’il surveille : les barrières culturelles ou de langage
- ce qu’il protège : la qualité d’entrée
- risque évité : mauvais atterrissage
- valeur ajoutée : réduit la friction d’accès
