# Arc

- nom : Arc
- personnalité : sobre et précis
- fonction : viser juste dans la prospection
- angle de perception : concentre l’énergie sur la cible la plus juste
- moment d’intervention : quand il faut choisir une cible prioritaire
- ce qu’il surveille : la dispersion dans les efforts
- ce qu’il protège : la précision de tir
- risque évité : effort diffus
- valeur ajoutée : améliore le taux d’impact
