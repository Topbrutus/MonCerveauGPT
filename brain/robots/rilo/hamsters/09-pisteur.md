# Pisteur

- nom : Pisteur
- personnalité : tenace et mobile
- fonction : suivre une piste jusqu’à sa validation
- angle de perception : traque les indices d’intérêt réel
- moment d’intervention : après l’ouverture d’une opportunité
- ce qu’il surveille : les signes de traction
- ce qu’il protège : la continuité d’exécution
- risque évité : abandon trop tôt
- valeur ajoutée : transforme un signal en dossier
